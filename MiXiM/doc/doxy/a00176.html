void phyPER::initialize ( int  stage) [protected, virtual]

Basic initialization for all modules.

Subscription should be in stage==0, and firing notifications in stage==1 or later.

NOTE: You have to call this in the initialize() function of the inherited class!

Reimplemented from BaseModule.

References BaseLayer::catPacketSignal, DeciderUWBIRED::catUWBIRPacketSignal, and BaseModule::findHost().

{
  BaseModule::initialize(stage);

    findHost()->subscribe(BaseLayer::catPacketSignal.initialize(), this);
    findHost()->subscribe(DeciderUWBIRED::catUWBIRPacketSignal.initialize(), this);
    maiPER.setName("maiPER");
    maiPERnoRS.setName("maiPERnoRS");

    nbSyncSuccesses = 0;
