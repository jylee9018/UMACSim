Detailed Description

This class is the interface for a Decider to the BasePhyLayer.

The Decider can do the following things using it:

  • get the current simulation time
  • get the list of AirFrames that intersect with a specific time interval (to calculate SNR)
  • tell the BasePhyLayer to hand an AirFrame up to the MACLayer
  • tell the BasePhyLayer to send a control message to the MACLayer (to answer a ChannelSenseRequest)
