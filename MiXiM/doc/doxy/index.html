MiXiM API Reference

The MiXiM API Classes Reference can be browsed in several ways (see links at top of this page):

  • Modules -- The most important classes, grouped by functionality.
  • Classes
  • Alphabetical List -- Alphabetical list of all classes.
  • Class List -- List of all classes with brief descriptions.
  • Class Hierarchy -- Inheritance tree of all classes.
  • Class Members -- All documented class members in alphabetical order.
  • Files
  • File List -- List of all C++ files.
  • Directories -- Directory tree

The best place to get started with is Modules --> base - base MiXiM modules.
