Detailed Description

Provides initialisation for several AnalogueModels and Deciders from modules directory.

Knows the following AnalogueModels:

  • SimplePathlossModel
  • LogNormalShadowing
  • JakesFading

Knows the following Deciders

  • Decider80211
  • SNRThresholdDecider
