Detailed Description

Layer address handling helper function.

This class helps to make the layer addressing more generic. The type definition for L2, and L3 addresses should be used for address definitions to simplify the switching to other types.

Both types (LAddress::L2Type, LAddress::L3Type) will be used as keys in in STL containers.
